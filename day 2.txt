Difference between Browser JS(console) v Nodejs

1)
Brower JS :
Can interact with the DOM, or other Web Platform APIs like Cookies.
Node Js: 
Those do not exist in  Node.js 

2)
Brower JS :
Browsers have the document, window 
Node Js: 
Doesn't have the document, window and all the other objects that are provided by the browser. 

3)
Brower JS :
Doesn't have all the nice APIs that Node.js provides through its modules, like the filesystem access functionality.
Node Js: 
Node.js provides nice APIs through its modules

4)
Brower JS :
Cannot control the environment
Node Js: 
Node.js you control the environment. Where to deploy , which version of Node.js you will run the application.

5)
Brower JS :
Browsers are slow to upgrade, sometimes on the web you are stuck with using older JavaScript / ECMAScript releases.
Node Js: 
Can write all the modern ES6-7-8-9 JavaScript that your Node.js version supports.
